# Machine Learning Assignments

This repository contains various Machine Learning assignments and lab work completed using Google Colab. The assignments are exported as PDF documents containing both the code (Jupyter Notebook cells) and the corresponding output.

## Assignments Included

- **Assignment 1:** [ML-Assignment 1 - Colab.pdf](./ML-Assignment%201%20-%20Colab.pdf)
  - **Summary:** Introduction to basic data operations using Pandas. We learned how to read data, find missing values, check data types, and index/sort data to prepare for machine learning tasks.
- **Assignment 2:** [ML-Assignment 2 - Colab.pdf](./ML-Assignment%202%20-%20Colab.pdf)
  - **Summary:** Focused on exploratory data analysis (EDA). We learned how to compute summary statistics, plot feature distributions using histograms, and perform data cleaning and transformation on a car price dataset.
- **Assignment 3:** [Assignment 3 - Colab.pdf](./Assignment%203%20-%20Colab.pdf)
  - **Summary:** Data visualization techniques using Matplotlib. We learned how to create and interpret Scatter plots, Bar plots, Box plots, Pie charts, and Line charts to visually understand data relationships.
- **Assignment 5:** [Assignment 5 - Colab.pdf](./Assignment%205%20-%20Colab.pdf)
  - **Summary:** Real-world data analysis on a taxi fare dataset. We learned how to handle time-series data (pickup datetimes) and geographical coordinates for feature engineering.
- **Assignment 6:** [Assignment6 - Colab.pdf](./Assignment6%20-%20Colab.pdf)
  - **Summary:** Introduction to regression models. We learned how to apply Linear and Logistic Regression on the Pima Indians Diabetes dataset to predict continuous values and binary outcomes.
- **Assignment 8:** [Assignment 8 - Colab.pdf](./Assignment%208%20-%20Colab.pdf)
  - **Summary:** Deep dive into Logistic Regression for classification. We learned how to evaluate models using confusion matrices and classification reports on a cosmetics shop dataset.
- **Assignment 9:** [Assignment 9 .ipynb - Colab.pdf](./Assignment%209%20.ipynb%20-%20Colab.pdf)
  - **Summary:** Unsupervised learning with K-Means clustering. We learned how to use the Elbow Method to find the optimal number of clusters for customer segmentation in a mall dataset.
- **Assignment 10:** [Assignment 10 .ipynb - Colab.pdf](./Assignment%2010%20.ipynb%20-%20Colab.pdf)
  - **Summary:** Predictive modeling with Linear Regression. We learned how to predict temperature trends over time and evaluate the model using Mean Squared Error (MSE) and R-squared metrics.
- **Assignment 11:** [Assignment 11 .ipynb - Colab.pdf](./Assignment%2011%20.ipynb%20-%20Colab.pdf)
  - **Summary:** Building decision trees. We learned how to train a Decision Tree Classifier to predict college admission chances based on GRE, TOEFL, and CGPA scores.
- **Assignment 13:** [Assignment 13.ipynb - Colab.pdf](./Assignment%2013.ipynb%20-%20Colab.pdf)
  - **Summary:** Advanced clustering applications. We applied K-Means clustering on the classic Iris dataset, evaluating the cluster quality and understanding feature groupings.

## Overview
These assignments cover various machine learning topics, algorithms, and practical implementations using Python libraries such as Scikit-Learn, Pandas, NumPy, and Matplotlib/Seaborn.
